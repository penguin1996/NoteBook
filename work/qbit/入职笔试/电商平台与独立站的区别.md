|      | 电商平台               | 独立站                                                       |
| ---- | ---------------------- | ------------------------------------------------------------ |
| 规则 | 限制多，且规则会变化   | 自由度高                                                     |
| 流量 | 平台自带流量           | 没有基础流量，需要自己大力推广来引流                         |
| 客户 | 客户是属于平台的       | 客户是属于自己的                                             |
| 比价 | 平台内存在严重的价格战 | 没有价格竞争的存在                                           |
| 利润 | 平台交易佣金8%~15不等  | 多数独立站无佣金，卖家可将更多资金用于独立站运行周转，将独立站的优势和利益最大化 |
